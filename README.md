# LOL-Without-Wegame

🔧一个轻量级的工具，它可以让 英雄联盟（国服） 绕过Wegame直接使用原生启动器登录游戏。

## 功能
- 一键删除并锁定 WeGame 临时文件，防止后续更新继续下载Wegame劫持文件
- 还原原生启动器登录体验
- 自动生成桌面快捷方式

## 📥 下载与安装

1. 前往本项目的 [Releases](https://github.com/Blur2x/LOL-Without-Wegame/releases) 页面。
2. 在最新版本中，下载 `lol-without-wegame.exe` 文件。
3. 将下载的程序放置在任意目录即可，无需安装。

## 🛠️ 使用说明

1. 建议**以管理员身份运行**本程序（由于需要锁定文件权限，普通模式可能会拦截失败）。
2. 选择好 英雄联盟 的目录，一定是根目录的文件夹，里面是存在TCLS这个文件夹的。参考路径（X：英雄联盟/TCLS），选取到英雄联盟这个文件夹就可以了。
   暂时不打算加入全盘扫描自动发寻找路径功能，毕竟要使用这个功能的人一定会找到路径，连路径都不会找的人也不建议使用。
3. 等待提示成功后，回到桌面，使用新生成的快捷方式启动游戏即可。
  （不要使用原来的 英雄联盟Wegame版 名称的图标，这个是被劫持了的。使用新生成的 原生LOL启动 图标）
